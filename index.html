The free, RESTful certificate authority

otoh.io is a free, open source X.509 certificate authority. It is written in Python and uses the CherryPy framework. Both the API and the Casandra persistence tiers can be used as a service at https://api.otoh.io, or can be run locally as Docker containers.

This site is under development. Check back soon for updates, or contribute at https://github.com/otohdotio.
